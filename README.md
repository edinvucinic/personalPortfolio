## Personal Portfolio

Welcome to the repository for my personal portfolio website! This website serves as a showcase of my projects, skills, and experiences.

About
This portfolio website is built using HTML, CSS, JavaScript, React, and Next.js. It features:

Portfolio Showcase: Highlighting my projects, skills, and accomplishments.
Interactive Design: Engaging and user-friendly interface to navigate through different sections.
Responsive Layout: Ensuring a seamless experience across various devices.

Technologies Used
Frontend: HTML, CSS, JavaScript, React, Next.js

Build Scripts: npm run build for production build, npm run dev for development server

Running the App
To start the development server and view the website locally:

Clone the repository:https://github.com/edinvucinic/personalPortfolio.git

Navigate to the project directory:
cd personal-portfolio

Install dependencies:
npm install

Start the development server:
npm run dev

Access the website:
Open your browser and go to http://localhost:3000 to view the website.
